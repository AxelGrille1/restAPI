# REST API Project

## Overview

This project is a RESTful API designed to help users understand how APIs work in practice. It is linked to a database and allows for the management of user data, specifically user names and emails. The API supports all CRUD (Create, Read, Update, Delete) operations.

## Features

- **Create**: Add new users with their email addresses.
- **Read**: Retrieve user information from the database.
- **Update**: Modify existing user details.
- **Delete**: Remove users from the database.

## Technologies Used

- Python
- Flask
- SQLAlchemy (for database interactions)
- SQLite (or another database of your choice)
